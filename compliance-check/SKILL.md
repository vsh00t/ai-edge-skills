# Compliance Check Skill

## Overview
Interactive compliance checker with built-in checklists for major cybersecurity and data protection standards. All checklists are hardcoded — no internet connection required.

## Supported Standards
- **ISO 27001 Annex A**: Key controls from Annex A
- **PCI-DSS v4.0**: All 12 requirements
- **Ecuador Ley de Protección de Datos**: Ecuadorian data protection law
- **Colombia Ley 1581/2012**: Colombian personal data protection law

## Usage
1. Select a compliance standard
2. Answer checklist questions (Yes / No / Partial / N/A)
3. Receive scored results with gap analysis

## Output
- Compliance percentage score
- Identified gaps
- Priority recommendations
- Maturity level assessment
