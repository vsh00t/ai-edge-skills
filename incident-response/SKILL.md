# Incident Response Skill

## Overview
Generates interactive incident response playbooks based on NIST SP 800-61 guidelines. Provides step-by-step procedures for 6 common incident types with progress tracking.

## Supported Incident Types
1. **Ransomware** — Malware encryption incident
2. **Data Breach** — Unauthorized data access/exfiltration
3. **Phishing Campaign** — Coordinated phishing attacks
4. **DDoS Attack** — Distributed denial of service
5. **Insider Threat** — Malicious insider activity
6. **APT/Advanced Attack** — Advanced persistent threat

## Playbook Phases (per NIST SP 800-61)
1. Preparation
2. Identification
3. Containment
4. Eradication
5. Recovery
6. Lessons Learned

## Output
- Current phase actions with checklist items
- Communication templates
- Evidence collection guidance
- Progress tracking per phase
